## 技术栈 | Tech Stack

- **前端框架 | Frontend:** Next.js 14 (App Router)
- **样式 | Styling:** Tailwind CSS
- **语言 | Languages:** TypeScript
- **部署 | Deployment:** Vercel

## 快速开始 | Quick Start

```bash
# 安装依赖 | Install dependencies
pnpm install

# 启动开发服务器 | Start dev server
pnpm dev

# 构建生产版本 | Build for production
pnpm build

# 启动生产版本 | Run production build
pnpm start
```
